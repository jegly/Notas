# Fonts for Notas

Place DotGothic16-Regular.ttf in this folder before building.

## Download DotGothic16

1. Go to: https://fonts.google.com/specimen/DotGothic16
2. Click "Download family"
3. Extract the zip
4. Copy `DotGothic16-Regular.ttf` to this folder

## Alternative: Command line download

```bash
wget -O DotGothic16-Regular.ttf "https://github.com/nickshanks/DotGothic16/raw/main/fonts/ttf/DotGothic16-Regular.ttf"
```

Or from Google Fonts:
```bash
wget "https://fonts.gstatic.com/s/dotgothic16/v17/v6-QGYjBJFKgyw5nSOqouUENAxX4vSRk.ttf" -O DotGothic16-Regular.ttf
```
